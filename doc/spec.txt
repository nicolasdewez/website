Partie technique :

* PHP 7.0 ou 7.1 (à priori 7.1)
* Symfony 3.2
* MySQL 5.5
* JQuery
* Bootstrap
* Docker pour les devs


Organisation des fichiers :

* Css et Js doivent être séparés par partie. Pas de Js de la partie admin sur le site et inversement.
* Découpage technique au max : services Sf, objets Js.


Styles :

* Pour la partie non sécurisée, s'inspirer de https://startbootstrap.com/template-overviews/clean-blog/
* Pour la partie sécurisée, bootstrap classique
* Formulaire de login un peu à la "grav"

Production :

* La partie sécurisée sera derrière une url différente de /admin (/secured-area ?)
* De la même façon le formulaire de login ne sera pas accessible via l'url /login (/access-secured-area ?)
* Le déploiement devra être fait de manière sécurisé : pas d'app_dev.php, ni de config.php etc.
* L'autoload devra être optimisé pour la production
* Procédure de déploiement, automatique ou manuelle ?
